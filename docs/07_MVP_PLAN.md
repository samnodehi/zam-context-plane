# 07 MVP Plan

## Goal
Build a CLI-only, independent MVP of the Context Control Plane.

## Scope
- **In Scope:** CLI tool, static JSON input reading, prompt plan generation, basic rule-based selectors, output JSON plan.
- **Out of Scope:** UI, live LLM calls, OpenClaw mutation, live agent history execution, runtime adapters.

## Phases
1. **Phase 1: CLI Skeleton & Schemas**
2. **Phase 2: Static Input Parsers**
3. **Phase 3: Basic Selectors & Budgeting**
4. **Phase 4: Output Generator & Trace Logs**
5. **Phase 5: Automated Evaluations on Fixtures**
