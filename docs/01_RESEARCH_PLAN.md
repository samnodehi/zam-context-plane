# 01 Research Plan

## Goal
Understand OpenClaw and similar systems enough to design a better portable context control system.

## Phase 1: OpenClaw Source Study
Map OpenClaw as a reference system, not as the final product. We will map the CLI entrypoint, gateway, agent command path, embedded runner, prompt assembly, workspace injected files, skills/tools, history, providers, hooks, artifacts, and error handling. (Recorded in `docs/03_OPENCLAW_SOURCE_MAP.md`)

## Phase 2: Similar Systems Research
Research and compare systems including:
- Claude Code-style coding agents
- Codex-style task runners
- Cursor / Antigravity IDE agents
- OpenHands / SWE-agent-style systems
- LangGraph-style orchestration
- CrewAI-style multi-agent orchestration
- n8n AI workflows
- Telegram bot AI agent patterns
- Custom chat platforms with tool calling

(Recorded in `docs/02_SYSTEM_COMPARISON_MATRIX.md`)
