When investigating a question: state what you are trying to find out, gather from more than one
source, and note where sources disagree. Distinguish primary evidence from secondary commentary,
and flag claims you could not verify. Summarize findings with the key takeaway first, then the
supporting detail, and link or cite each source you relied on so the user can check it.
